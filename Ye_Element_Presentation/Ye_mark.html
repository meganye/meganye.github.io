Overview

According to the Mozilla Developer Network: "The HTML Mark Element (<mark>) represents highlighted text, i.e., a run of text marked for reference purpose, due to its relevance in a particular context. For example it can be used in a page showing search results to highlight every instance of the searched-for word."

Usage

  • Used to highlight text
  • Should not be used for syntax highlighting (use <span> instead)
  • Not to be confused with the <strong> element (the <mark> element is used to highlight text of relevance, not importance)
  • Includes the global attributes
